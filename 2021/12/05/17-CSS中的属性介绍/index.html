17.CSS中的属性介绍

Author: zjy Date: 十二月 5, 2021  9:06:54

圆角属性

以前在CSS3出现之前，我们如果想要在页面上实现圆角效果，需要使用对应的背景图来实现。在修改圆角时需要设计师修改对应的图片发给我们，沟通成本很高，所以在CSS3中才有圆角概念。


实际上的圆角，就是一个半径为设置的圆角值的圆形放在对应边角的效果。
设置圆角如果使用的是一个值，则表示四个角都是圆角，如果设置四个值分别代表左上，右上，右下，左下四个方向。
如果是两个值，则表示（左上 右下） （右上 左下）
如果我们要给元素的一个角设置两个值，则会是一个椭圆效果


单独定义角

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-left-radius
  • border-bottom-right-radius

定义一个值是普通圆角，两个值是椭圆圆角

阴影

阴影指元素上的阴影效果，分别为box-shadow和text-shadow

box-shadow

我们只需要会设置即可

x值的作用实现阴影x轴偏移，为正值时向右偏移，为负值时向左偏移
y值的作用实现阴影y轴偏移，为正值时像下偏移，为负值时向上偏移
模糊程度 表示阴影的模糊效果，一般使用阴影都会使用模糊，因为模糊更好看
阴影大小 默认阴影大小和我们对应的元素一样大，设置大小，会让阴影变得更大。
inset 添加内阴影 写不写inset

外阴影

内阴影

写多个阴影

如果我们想要实现多个阴影怎么办，使用逗号隔开每个阴影的属性即可。

0 8px 16px rgb(0 0 0 / 18%)

text-shadow

没有内阴影，也没有模糊大小，只有x y 模糊程度 颜色 四个值


背景渐变色

CSS中渐变色不属于颜色概念，属于图片概念。所以渐变色需要使用background-image属性进行设置而不是background-color

线性渐变

方向

  • to 方向

top \ right \ bottom \ left \ top right \ top left \ bottom right \ bottom left

  • 角度

角度单位是deg
0度指to top，正角度顺时针旋转，负角度逆时针旋转

颜色及宽度的设置

在线性渐变中，我们可以写很多很多很多的颜色
linear-gradient(方向, 颜色1, 颜色2, 颜色3, 颜色4, 颜色n)
并且可以在颜色的后面设置这个颜色占据的空间
linear-gradient(方向, 颜色1 值, 颜色2 值, 颜色3 值, 颜色4 值, 颜色n 值)
值可以是固定值也可以是百分比，表示对应的元素从哪个位置开始到下一个颜色的位置结束进行渐变。



重复线性渐变

在我们添加了渐变宽度的基础上，实现重复效果

径向渐变

  • shape

ellipse (默认): 指定椭圆形的径向渐变。
circle ：指定圆形的径向渐变

  • size

farthest-corner (默认) : 指定径向渐变的半径长度为从圆心到离圆心最远的角
closest-corner ： 指定径向渐变的半径长度为从圆心到离圆心最近的角
farthest-side ：指定径向渐变的半径长度为从圆心到离圆心最远的边
closest-side ：指定径向渐变的半径长度为从圆心到离圆心最近的边

  • size

表示圆心位置 可以设置两个值分别是x和y的值 50% 50% 表示正中心。
